# 129 V17 Promotion-Evidence Protocol V1

## Scope
- freeze the first bounded protocol for stating what new evidence would be required to change retained-feature promotion judgment for the current provisional candidates

## Core Rules
- promotion evidence is not promotion
- no strategy integration is allowed
- no local-model opening is allowed
- no wide replay or refresh expansion is allowed
- the protocol only states shortfalls and admissible proof types

## Review Axes
- sample_breadth_gap
- cross_pocket_or_cross_regime_gap
- non_redundancy_stress_gap
- safe_consumption_beyond_report_only_gap

## Expected Outputs
- per-feature promotion shortfall rows
- admissible proof types for each feature
- a clear bounded next step that can change promotion judgment without violating current guardrails
