# V1.12Z CPO Bounded Cycle Reconstruction

`V1.12Z` is the first true downstream experiment after the CPO foundation line was cleaned and challenged.

The point is not to start training.
The point is to use the cleaned foundation to reconstruct one bounded optical-link cycle and see:

- whether the catalyst and board chronology really line up
- whether role transitions still make sense when placed into one cycle map
- whether mixed-role and spillover ambiguity still survives in a visible way

This phase is explicitly ambiguity-preserving.
