# 113 V14 Context Consumption Protocol V1

## Purpose
- Freeze how bounded catalyst and concept context may be consumed inside `V1.4`.

## In Scope
- consumable input set
- binding-scope rules
- report-only context feature families
- explicit forbidden actions

## Out Of Scope
- strategy integration
- retained-feature promotion
- new replay or wide ingestion
- formal model work

## Success Criteria
- define bounded and auditable context-consumption rules
- keep the branch replay-independent
- prepare the next bounded schema-level artifact
