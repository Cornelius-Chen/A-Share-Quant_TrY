# V1.12R Adjacent Cohort Validation

`V1.12R` is the first precise cleaning pass after the CPO registry and schema hardening work.

It does not try to finish the whole CPO line. It only answers:

- which adjacent names are already clean enough to preserve as bounded review assets
- which names still belong to over-broad role clusters
- which names should stay pending until later role or branch splits are cleaner

This phase is still review-only. It does not open training.
