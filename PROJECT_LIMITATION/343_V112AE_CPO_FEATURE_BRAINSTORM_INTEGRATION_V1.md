## Most Important Reading
- The next missing layer is not “more generic indicators”.
- The next missing layer is:
  - time geometry
  - role handoff
  - false diffusion and weak-cohort migration
  - late-cycle maturity and spillover separation

## Important Discipline
- None of these are formal features yet.
- Some are explicitly overlay-only.
- Several cannot be operationalized cleanly until:
  - breadth formula is frozen
  - turnover normalization is frozen
  - name/alias/concept-tag structure exists
  - some microstructure layer exists
