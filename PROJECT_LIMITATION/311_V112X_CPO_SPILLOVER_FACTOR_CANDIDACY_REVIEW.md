# V1.12X CPO Spillover Factor Candidacy Review

`V1.12X` exists because the CPO line intentionally preserved noisy A-share spillover rows instead of dropping them.

This phase does not train on them.
It does not promote them into formal features.
It uses a bounded black-box sidecar posture to ask:

- do any of these preserved rows deserve bounded review-only factor candidacy status?
- and which rows should remain weaker memory-only assets?
