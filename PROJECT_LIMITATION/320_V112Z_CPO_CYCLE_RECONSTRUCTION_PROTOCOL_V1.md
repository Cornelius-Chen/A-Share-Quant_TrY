# V1.12Z CPO Cycle Reconstruction Protocol V1

`V1.12Z` freezes a bounded reconstruction protocol before any reconstructed map is claimed.

The protocol requires:

- stage ordering
- catalyst ordering
- core / adjacent / branch role transitions
- board chronology overlays
- spillover overlays

It also explicitly forbids:

- training
- signal generation
- automatic feature promotion
- silent removal of mixed-role or spillover ambiguity
